PANDUAN SINGKAT
1. Deploy Code.gs di Google Apps Script sebagai Web App.
2. Copy URL hasil deploy.
3. Buka file script.js, ganti API_URL dengan URL Apps Script.
4. Upload index.html, style.css, script.js ke GitHub repo.
5. Aktifkan GitHub Pages di Settings > Pages > Deploy from branch.

Catatan:
- Jika posisi kolom di sheet bergeser, sesuaikan index pada Code.gs.
- File Code.gs dipakai di Apps Script, bukan di GitHub Pages.
